About Me

I'm a Web Developer

Yahya is a skilled Frontend Developer and UI/UX Designer with a strong passion for creating visually stunning and highly responsive web experiences. With expertise in HTML, CSS, and JavaScript, he crafts sleek and interactive designs that enhance user engagement. His deep understanding of Bootstrap and React.js allows him to develop dynamic, scalable, and efficient web applications. As a Web Designer, Yahya focuses on building aesthetically pleasing layouts with seamless navigation. His expertise in animation design using GSAP enables him to create smooth, eye-catching animations that bring websites to life. His designs are not only visually appealing but also functional, ensuring a responsive user experience across all devices. Beyond frontend development, Yahya is proficient in Node.js and MongoDB, making him a full-fledged MERN stack developer. His ability to manage both the frontend and backend aspects of web applications ensures smooth data flow and seamless integration between user interfaces and databases. With a keen eye for detail and a passion for innovation, Yahya continuously pushes the boundaries of modern web development. His commitment to delivering high-quality, user-centric solutions makes him a valuable asset in the ever-evolving world of web technologies.
